# ShopLite - Phiên bản 1: HTML & CSS cơ bản

Dự án ShopLite là một ứng dụng web bán hàng tối giản được xây dựng theo từng bước hướng dẫn.
Phiên bản này tập trung vào việc dựng khung giao diện sử dụng HTML Semantic và định dạng CSS cơ bản (Reset, Typography, Layout cơ bản).

## Cấu trúc thư mục
* `index.html`: Khung trang danh sách sản phẩm.
* `style.css`: Các tùy chỉnh định dạng nền tảng và reset CSS.
