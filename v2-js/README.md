# ShopLite - Phiên bản 2: JavaScript cơ bản

Dự án ShopLite phát triển ứng dụng bán hàng.
Phiên bản này bổ sung tệp dữ liệu giả `data.js` và các hàm bổ trợ xử lý danh sách sản phẩm (`helpers.js`).

## Các tệp tin
* `index.html`, `product.html`, `cart.html`: Giao diện tĩnh ban đầu.
* `style.css`: Hệ thống thiết kế dùng CSS variables.
* `data.js`: Chứa danh sách mảng sản phẩm mẫu tĩnh.
* `helpers.js`: Các hàm xử lý mảng sản phẩm (`filterByKeyword`, `sortByPrice`).
